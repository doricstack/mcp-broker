## Usage

Use `mark-favicon.svg` for small contexts from 16x16 to 64x64.
Use `mark.svg` for app icons, GitHub avatars, and square placements.
Use `horizontal.svg` for README headers, docs, and websites.
Use one-color assets when the mark needs to sit on uncontrolled backgrounds.
