# mcp-broker brand assets

This folder contains the selected production brand assets for `mcp-broker`.

The brand direction is routing backbone: one local MCP entry routed to many upstream MCP servers, with profile gates and safe tool exposure.

## Layout

- `assets/`: production SVG and PNG assets.
- `brand-overview.png`: visual overview board for the selected direction.

## Recommended files

- GitHub avatar or app icon: `assets/app-icon-1024.png`
- Favicon: `assets/favicon-32.png`
- README header: `assets/readme-header.svg`
- Social preview: `assets/github-social-preview.svg`
- Primary mark: `assets/mark.svg`
- Favicon-safe mark: `assets/mark-favicon.svg`
- Horizontal wordmark: `assets/horizontal.svg`
- One-color marks: `assets/mark-black.svg` and `assets/mark-white.svg`
- Design tokens: `assets/tokens.css` and `assets/tokens.json`

## Rules

- Use the files under `assets/` for public surfaces.
- Prefer SVG for docs and websites.
- Use PNG exports only where registries or social platforms do not accept SVG.
- Do not reintroduce alternate concept folders into the public repo.
